# Fullstack ToDo App

Ez egy **Fullstack ToDo alkalmazás** React (Vite + TypeScript) frontenddel és Node.js (Express + TypeScript) backenddel, Prisma MySql adatbázissal.

# Technológiák:

# Frontend

React Vite
TypeScript
Bootstrap 5
React Hook Form
Zod
Axios API

# Backend

Node.js + Express + TypeScript
Prisma + MySql
Zod
CORS
dotenv
