# Fullstack ToDo App

Ez egy **Fullstack ToDo alkalmazás** React (Vite + TypeScript) frontenddel és Node.js (Express + TypeScript) backenddel, Prisma Sqlite adatbázissal.

# Technológiák:

# Frontend

React Vite
TypeScript
Bootstrap 5
React Hook Form
Zod
fetch API

# Backend

Node.js + Express + TypeScript
Prisma + Sqlite
Zod
CORS
dotenv
