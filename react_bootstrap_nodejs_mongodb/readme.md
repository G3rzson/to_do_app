# Fullstack ToDo App

Ez egy **Fullstack ToDo alkalmazás** React (Vite + TypeScript) frontenddel és Node.js (Express + TypeScript) backenddel, MongoDB adatbázissal.

# Technológiák:

# Frontend

React Vite
TypeScript
Bootstrap 5
React Hook Form
Zod
fetch API

# Backend

Node.js + Express + TypeScript
MongoDB + Mongoose
Zod
CORS
dotenv
